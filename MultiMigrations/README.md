# Entity framework Multi Migrations

엔트리 프레임워크에서(Entity framework) 여러 DB를 관리하기위한 마이그레이션을 구현하는 클래스 라이브러리입니다.

이 프로젝트를 복사하거나 프로젝트의 내용만 복사하여 다른 프로젝트에 사용할 수 있도록 구성하였습니다.

자신의 프로젝트에 맞게 참조와 코드를 수정하는 것이 좋습니다.

<br />

Entity framework 6
